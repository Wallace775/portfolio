# Internacionalização (i18n) do Portfólio

Este documento descreve o sistema de internacionalização implementado no portfólio de Wallace Phellipe.

## Funcionalidades

- Suporte para múltiplos idiomas (atualmente português e inglês)
- Detecção automática de idioma do navegador
- Persistência da preferência do usuário
- Atualização dinâmica da interface
- Sistema modular para fácil adição de novos idiomas

## Estrutura de Arquivos

```
i18n.js            # Implementação do sistema de internacionalização
```

## Idiomas Suportados

- Português (padrão) - `pt`
- Inglês - `en`

## Como Funciona

### 1. Traduções

As traduções são armazenadas no objeto `translations` dentro da classe `I18n`:

```javascript
translations = {
    pt: {
        'contact-form-title': 'Envie uma Mensagem',
        'contact-name': 'Nome',
        // mais traduções...
    },
    en: {
        'contact-form-title': 'Send a Message',
        'contact-name': 'Name',
        // mais traduções...
    }
};
```

### 2. Uso no HTML

Elementos que precisam de tradução recebem o atributo `data-i18n`:

```html
<h3 data-i18n="contact-form-title">Send a Message</h3>
```

### 3. Tipos de Atributos de Tradução

- `data-i18n` - Texto do elemento
- `data-i18n-placeholder` - Placeholder de inputs
- `data-i18n-title` - Atributo title
- `data-i18n-alt` - Atributo alt de imagens

### 4. Mudança de Idioma

O idioma pode ser alterado programaticamente:

```javascript
i18nInstance.setLanguage('en'); // Define idioma como inglês
```

## Adicionar Novo Idioma

1. Adicione o código do novo idioma ao array `supportedLanguages`:

```javascript
this.supportedLanguages = ['pt', 'en', 'es']; // Adicionando espanhol
```

2. Adicione as traduções no objeto `translations`:

```javascript
translations = {
    // ... idiomas existentes
    es: {
        'contact-form-title': 'Envíe un Mensaje',
        'contact-name': 'Nombre',
        // mais traduções...
    }
};
```

## Seletor de Idioma

O sistema automaticamente adiciona um seletor de idioma no cabeçalho da página. Este seletor:
- É posicionado no canto superior esquerdo
- Permite troca de idioma com um clique
- Salva a preferência no localStorage

## Persistência

A preferência de idioma do usuário é salva no `localStorage` e restaurada em visitas subsequentes.
